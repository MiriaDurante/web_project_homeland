# Homeland

Homeland é uma página web que apresenta cidades natais de colaboradores da comunidade TripleTen ao redor do mundo. O projeto reúne descrições informativas e imagens organizadas em seções estruturadas, com foco em clareza visual e organização do conteúdo.

Este foi o meu primeiro projeto desenvolvido com abordagem responsiva, adaptado para diferentes tamanhos de tela.

## Responsividade

O layout foi projetado para três resoluções principais:

- Desktop: 1280px
- Tablet: 768px
- Celular: 320px (sendo a menor resolução criada até o presente momento)

A adaptação entre os tamanhos de tela foi planejada para preservar a legibilidade do conteúdo e a consistência da estrutura visual, para facilitar a navegação e leitura dos textos e a visualização de cada imagem da galeria.

## Estrutura da Página

O site é composto por:

- Cabeçalho com identidade visual do projeto
- Seção de destaque com título, subtítulo e imagem principal
- Seção introdutória com contextualização do conteúdo
- Galeria de imagens
- Seções individuais com descrição de cidades (sendo a última de escolha pessoal, e escolhida fora do design Figma)
- Rodapé com identificação de autoria

## Tecnologias Utilizadas

- HTML5 para estruturação semântica do conteúdo
- CSS para estilização e construção do layout
- Grid para organização estrutural de elementos
- Flexbox para alinhamento e distribuição de componentes
- Efeitos de hover e shadow para refinamento visual da interface

## Organização do CSS

A estilização foi estruturada de forma modular, com separação por blocos correspondentes às seções da página.  
Os arquivos de estilo foram organizados por meio de @import, permitindo maior organização e manutenção do código.

## Processo de Desenvolvimento

O projeto foi desenvolvido a partir de um design fornecido no Figma pela TripleTen.  
A implementação do código foi realizada integralmente de forma individual.

## Melhorias Futuras

Evoluções planejadas para o projeto:

- Ampliação do suporte responsivo para resoluções inferiores a 320px
- Refinamento do comportamento adaptativo do layout em diferentes breakpoints
- Aprimoramento da consistência visual da interface
- Otimização da estrutura e organização dos estilos CSS
